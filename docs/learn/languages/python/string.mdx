# String

String is a collection of items of same data type stored at contiguous memory locations.

~~(deja vu enough?)~~ Well, **string is array**.

**Python strings are "immutable"** which means they cannot be changed after they are created.

## Declare

To declare a string, you can assign variable to empty string

```python3
// -highlight-start
s = ''
// -highlight-end
```

:::tip

You can use `s = ''` to delete all characters of the string

:::

## Basics

Since, string is array, it has array's basic function.

### Get array size

Use `len(s)`

```python3
s = 'Python'
// -highlight-start
len(s)
// -highlight-end


// highlight
> s: 6
```

### Check empty string

Check the `len`.

```python3
s = 'Python'
// -highlight
if len(s) == 0:
  print('Nothing is here!')
else:
  print('String is not empty!')
```

### Insert

Use slicing array. Common pattern is **`str[:pos] + something + str[pos:]`**

```python3
s = 'Python'
s2 = ' is great!'
// -highlight-start
s = s[:6] + s2 + s[6:]
// -highlight-end


// highlight
> s: 'Python is great'
```

### Delete

Same as [insert](#insert), use slicing array to delete. Common pattern is **`str[:start] + str[end:]`**

```python3
s = 'Python is great'
// -highlight
s = s[:6] + s[14:]

// highlight
> s: 'Python'
```

### Find

Use **`find(string)`**. You can user `find(string, start, end)` to find in a specific range.

```python3
s = 'Python'
// -highlight-start
f = s.find('t')
nf = s.find('z')
r = s.find('t', 0, 1)
// -highlight-end


// highlight-start
> f: 2
> nf: -1
> r: -1
// highlight-end
```

### Get sub-string

Use **`str[start:end]`**

```python3
// -highlight
'Python'[1:3]


// highlight
> yt
```

### Concat

Use `+` operator.

```python3
// -highlight
'Python ' + 'is great' + '!'


// highlight
> Python is great!
```

## Validation of string

### ASCII tables

Lookup for ASCII table [here](https://www.lookuptables.com/text/ascii-table) or [simply here](http://www.asciichars.com/_site_media/ascii/ascii-chars-landscape.pdf).

### Check if character is a digit

You can use `ord(char)` to get ASCII decimal number of a character.

Based on the ASCII table, you can see **digits** will have decimal number from `48`, which is 0 to `57`, which is 9.

```python3
// -highlight
if ord(str[0]) >= 48 and ord(str[0]) <= 57:
    # it is a digit, do something
```

:::tip
Remember number for ASCII '0' is **48**.

Then, you know there is digits from 0 to 9, add more up to 9, which come up to 57.
:::

You can use `chr(number)` to convert ASCII code to a character.

```python3
// -highlight
zero = chr(48)


// highlight
> 0
```
