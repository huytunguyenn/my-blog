# Stack & Queue

## Stack

LIFO (Last In First Out)

```bash
 0  1  2
[4, 7, 8]

| 8 | 2
| 7 | 1
|_4_| 0
```

### Declare

<Tabs groupId="stack">


</TabItem>
<TabItem value="java" label="Java">


</TabItem>
<TabItem value="cpp" label="C++">

```cpp

using namespace std;
```

</TabItem>
</Tabs>

## Queue

FIFO (First In First Out)

```bash
   _________
=>  4 3 2 1  =>
   ---------
```

### Declare

<Tabs groupId="queue">
<TabItem value="py" label="Python">

```py

q = Queue()
```

</TabItem>
<TabItem value="java" label="Java">

```java

import java.util.Queue;

Queue<Integer> q = new LinkedList<Integer>();
```

</TabItem>
<TabItem value="cpp" label="C++">

```cpp

using namespace std;
```

</TabItem>
</Tabs>
