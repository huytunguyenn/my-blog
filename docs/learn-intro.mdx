## Fast way

**Viết docs mới:**
- Tạo thư mục con trong docs
- Vào `sidebars.js` thêm type category. Ở phần `items` thì để
```react
{
  type: 'autogenerated',
  dirName: 'path/to/folder', // đường dẫn đến thư mục mới tạo (bắt đầu từ docs/)
},
```
- Viết docs bỏ vào thư mục, tự động render ra sidebar
